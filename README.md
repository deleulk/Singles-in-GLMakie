# Singles-in-GLMakie
This repository contains a folder which contains the files to play Singles. All the code in written in julia.
The guidelines are in dutch:
De volgende stappen leggen uit hoe u het spel kunt spelen met mijn code.
Voor u aan de handleiding begint is het aangeraden om Mircosoft VsCode en
alle bijhorende packages te installeren in dat programma. U zult in Mircosoft
VsCode bij extensions de juiste extension moeten installeren voor Julia. Let
op: het programma zal misschien niet meteen uw installatie van Julia vinden
op uw harde schijf. Het is daarom belangrijk om zelf de directory naar uw eigen
installatie van Julia aan te passen. Wanneer u julia ge¨ınstalleerd heeft, maakt
u een testbestand aan en opent u de repl met ”ALT + J” en ”ALT + O”. In
de repl typt u een gesloten vierkant haakje. Dan typt u add pkg GLMakie en
add pkg Random. U kunt nakijken of deze al ge¨ınstalleerd zijn door status in
dezelfde repl in te typen.
1. Nu heeft u alle benodigdheden ge¨ınstalleerd op uw computer en bent u
klaar om het spel te spelen.
2. Klik op de hyperlink in de bijlage en download de twee bestanden die u
op die pagina vindt: "singles_finaal.jl" en "singles_bot.jl".
3. Wanneer dit gelukt is zorgt u dat ze in dezelfde folder zitten. Vervolgens
opent u de folder in Microsoft VsCode
4. Dan rest u nog het bestand singles_bot.jl te runnen door rechtsboven
op ”run” te klikken
4
5. Veel speelplezier.
